# PRD: Simple Python Calculator

**Priority**: Low
**Complexity**: Simple
**Category**: coding
**Module**: general

## What to Build

Create a simple Python script that:
1. Defines two variables: a=10, b=20
2. Calculates their sum
3. Prints the result

## Files

- `simple_calculator.py`

## Expected Output

When run with `python simple_calculator.py`, it should print:
```
The sum of 10 and 20 is 30
```

## Success Criteria

1. File exists at correct location
2. Script runs without errors
3. Output matches expected format exactly
