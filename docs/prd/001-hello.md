# PRD: Hello World

**Priority:** Low
**Complexity:** Simple
**Category:** coding

## What to Build

Create a simple hello function in Go:
- File: `governor/cmd/tools/hello.go`
- Function: `SayHello(name string) string`
- Returns "Hello, {name}!" or "Hello, World!" if empty

## Success Criteria

- Compiles without errors
- Returns correct greeting format
