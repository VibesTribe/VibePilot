# Test PRD - Calculator Modulo

## Summary
Create a modulo function for the calculator module.

## Requirements
- Function: `modulo(a, b int) (int, error)`
- Returns remainder of a / b
- Returns error when dividing by zero

## Acceptance Criteria
- Returns correct remainder
- Handles division by zero error
- Unit tests included

## Technical Notes
- Place in `internal/calc/modulo.go`
- Tests in `internal/calc/modulo_test.go`
