# PRD: Hello World Function

**Priority:** Low
**Complexity:** Simple
**Category:** coding

## Context

Create a simple hello world function to test the complete VibePilot flow from PRD to completion.

## What to Build

Create a Go function that returns a greeting:
- File: `governor/cmd/tools/hello.go`
- Function: `SayHello(name string) string`
- Returns: "Hello, {name}!"

## Success Criteria

- Function compiles
- Returns correct format
- Handles empty name (returns "Hello, World!")
