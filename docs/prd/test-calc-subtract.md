# Test PRD - Calculator Subtract

## Summary
Create a simple subtract function for the calculator module.

## Requirements
- Function: `subtract(a, b int) int`
- Returns difference of two integers (a - b)
- Must handle positive and negative numbers

## Acceptance Criteria
- Returns correct difference
- Handles edge cases (zero, negative)
- Has unit tests with >80% coverage

## Technical Notes
- Place in `internal/calc/subtract.go`
- Tests in `internal/calc/subtract_test.go`
