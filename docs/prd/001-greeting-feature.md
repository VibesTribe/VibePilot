# PRD: Greeting Feature

## Overview
Add a simple greeting function to the VibePilot codebase.

## Requirements
1. Create a `greeting.go` file in the `governor/pkg/greeting` directory
2. Implement a `SayHello(name string) string` function that returns "Hello, {name}!"
3. Add a simple test file `greeting_test.go`

## Acceptance Criteria
- Function returns proper greeting format
- Tests pass
- Code follows Go conventions

## Priority
Medium

## Estimated Complexity
